About

I majored in Mandarin Chinese at Indiana University - Bloomington, and lived in China for a year teaching English to both improve my language and help others learn another language.

Once I came back, I deciced I wanted to study another skill, as knowing Mandarin was not in high demand in Indy, so I decided to attend Eleven Fifty Coding Academy to become a Software Developer.

Storefront Project

We were to make a static store front for a business of our choice. At the time I had been reading a few comics, so figured I should try to make a store front based on that. The Comic of the Month is Saga, and I also feature the brands Marvel, DC, and Image. Clicking the image will take you to the CodePen project.

Escape From Jail Game

We were to make a game using Visual Studio Community. I love story and adventure games, so I decided to make a text-adventure game that had a bit of story to it. Clicking on the image will take you to its GitHub page.

Rick and Morty CSS Project

We were to make a creature only using CSS. After brainstorming what to do, I figured Morty from the TV Show Rick and Morty would be a good idea. I also implemented and modified an earlier assignment of creating a smiley face, and molded it after Morty's face, and figured it'd make sense in that universe to have it floating underneath a saucer. Clicking on the image will take you to the CodePen project

Gold Badge Challenges

Star Wars API (Blue Badge Project)

For our Blue Badge Project, we were to create a database with a theme of our choosing. My two partners and I are fans of Star Wars, and decided it would be fun to do a project about Bounty Hunting, so users would be able to log hunters, ships, targets, weapons, and plants. Clicking the image will take you to the GitHub repository.

WrestleHeavy Project

For our Red Badge Project, we were to create a ASP.NET MVC. For this project I wanted to choose something I have a deep interest in, which happens to be pro-wrestling. Users are able to create a wrestler, promotion, and championship, and will eventually be able to simulate the odds when pitting two fighters together. Clicking the image will take you to the GitHub repository.
